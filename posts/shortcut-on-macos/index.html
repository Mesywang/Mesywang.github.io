macOS 环境快捷键

7 Mar 2018 on Developer, Mac
views
Updated 6 days ago

在 macOS 中，有些组合快捷键很实用，但是容易忘记。故记下，省去日后 Google 的流量费，早餐加个蛋。

为了精简描述，本文为以下按键采用缩写：

Command→Cmd

Control→Ctrl

笔者发誓这绝对不是偷懒不想码多几个字母。

macOS

用途快捷键
熄灭屏幕Shift+Ctrl+Power
截取全屏Shfit+Cmd+3
截取选择区域Shfit+Cmd+4
截取活动窗口/录像Shfit+Cmd+5
剪切文件Cmd+C → Alt+Cmd+V
光标跳到字符串头部Ctrl+A
光标跳到字符串尾部Ctrl+E
选中光标至字符串起始部分Shift+Ctrl+A
选中光标至字符串末端部分Shift+Ctrl+E
Finder 开启/隐藏 不可见文件(macOS 10.12 以上)Shfit+Cmd+.

Terminal

用途快捷键
当前路径打开新 tabCmd+T
左右切换 tabShfit+Cmd+←/→

Chrome

用途快捷键
恢复上次打开的所有 TabShfit+Cmd+T
删除搜索智能填充的记录↑/↓ 选中 → Fn+Shift+Delete
当前 tab 切换为 HomgPageShfit+Cmd+H
开启/关闭 DevToolsAlt+Cmd+I
跳到指定 TabCmd+num
左右切换 TabOpt+Cmd+←/→
全屏Ctrl+Cmd+F
全屏隐藏/显示 tabsShift+Ctrl+F
显示/隐藏书签栏Shift+Ctrl+B

Sublime Text

用途快捷键备注
打开文件Cmd+P 
选择高亮语法Shfit+Cmd+P 
文本编辑：跳到指定行Ctrl+G 
代码缩进美化Shift+Ctrl+H需要安装 HTML/CSS/JS Prettify 插件

IntelliJ IDEA

用途快捷键
定位文件Cmd+O
定位 ClassShfit+Cmd+O
重命名项目/模块Shift+F6
运行 ClassCtrl+R
Debug ClassCtrl+D
清除无效importAlt+Ctrl+O
缩进代码Alt+Ctrl+I
生成main函数psvm → Enter
生成System.out.println();sout → Enter

  • 编辑类

    用途快捷键
    选取光标至行头Shift+Cmd+←
    选取光标至行末Shift+Cmd+→
    格式化选中代码块Alt+Cmd+L
    编辑位置后退Alt+Cmd+←
    编辑位置前进Alt+Cmd+→
