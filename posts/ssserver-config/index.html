Shadowsocks Server配置

25 Jun 2018 on Developer, Shadowsocks
views
Updated 2 weeks ago

配置

服务端采用json文件存储配置信息，可分单用户和多用户两种情况：

单用户

{
  "server": "0.0.0.0",
  "server_port": 10000,
  "password": "mypasswd",
  "timeout": 300,
  "method": "aes-256-cfb"
}

多用户

{
  "server": "0.0.0.0",
  "port_password": {
    "1001": "password_a",
    "1002": "password_b"
  },
  "timeout": 300,
  "method": "aes-256-cfb"
}

启动命令

$ /bin/ssserver -c /path/to/config.json -d start

参数说明：

-c 表示加载指定配置文件

-d 表示后台模式
